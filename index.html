Single-Page Editable Movie Trailer Web Application Project

Goal

To build an API server and create a single-page web application that link to recieve YouTube videos. It also has the ability to create, read, update and delete (CRUD) a list of movie trailer data into the web application.

Purpose

The major benefit of the MEAN stack is that it's extremely quick to prototype with.
